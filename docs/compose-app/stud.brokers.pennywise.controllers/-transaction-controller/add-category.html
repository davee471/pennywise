suspend fun addCategory(name: String, iconName: String): Result<Unit>

Creates and persists a new user-defined category.

The category is saved with id = 0; the database assigns the real primary-key ID on insert.

Return

Result.Error with ErrorType.VALIDATION if name is blank; otherwise the result of the database insert operation.

Parameters

name

The display name of the category. Must not be blank.

iconName

The Material icon name string used to render this category's icon in the UI.
