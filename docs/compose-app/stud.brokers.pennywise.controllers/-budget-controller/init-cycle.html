initCycle

suspend fun initCycle(totalAmount: Double, start: LocalDate, end: LocalDate)

Creates and persists a new budget cycle, then reloads state.

Any previously active cycle should be reset via resetCycle before calling this, as the database is expected to hold at most one active cycle at a time.

Parameters

totalAmount

The starting monetary allowance for the cycle. Must be 0.

start

The inclusive start date of the cycle.

end

The inclusive end date of the cycle. Must be ≥ start.
