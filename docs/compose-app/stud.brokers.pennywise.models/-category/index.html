Category

@Serializable
data class Category(val id: Long, val name: String, val iconName: String)

Represents a transaction category used to classify expenses and income.

Categories can be either built-in defaults (e.g. "Food", "Transport") or user-defined. Each category is identified by a unique id and carries a display name and an iconName that maps to a drawable resource or icon identifier on the host platform.

Instances are stored in the database and embedded directly inside Transaction objects so that category metadata is preserved even if the category is later deleted by the user.
